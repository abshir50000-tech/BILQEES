# Bilqees - Bringing Muslims Closer to Allah

Welcome to **Bilqees**, a beautiful and user-friendly Islamic website dedicated to bringing Muslims closer to Allah through the study of the Quran, Hadith, and Islamic knowledge.

## 🌟 Features

- **Quran Section**: Daily Quranic verses with explanations and Tafsir
- **Hadith & Sunnah**: Authentic Hadith collections and teachings of Prophet Muhammad (PBUH)
- **Islamic Knowledge**: Comprehensive resources on the Five Pillars, Fiqh, Islamic history, and personal development
- **Daily Reminders**: Inspiring daily Islamic reminders to strengthen your faith
- **Prayer Times**: Track prayer times based on your location
- **Contact Form**: Connect with us for questions and suggestions

## 🎨 Design

The website features a serene Islamic design with:
- Deep green and gold color scheme representing Islamic values
- Responsive layout for mobile and desktop devices
- User-friendly navigation
- Beautiful cards and components

## 🚀 Getting Started

1. Clone this repository:
```bash
git clone https://github.com/abshir50000-tech/bilqees.git
```

2. Navigate to the project folder:
```bash
cd bilqees
```

3. Open `index.html` in your web browser to view the website.
